Tantangan Akhir: Data & Privacy App

Buat aplikasi MIT App Inventor bertema Smart Jajan atau Data Aman Sekolah. Aplikasi harus punya input, validasi, keputusan, dan pesan keamanan.

Wajib Ada

  • Minimal 2 input pengguna.
  • Validasi kosong dan angka.
  • Output berupa saran/keputusan.

Nilai Plus

  • Menggunakan PasswordTextBox untuk data rahasia.
  • Ada Notifier untuk pesan error.
  • Flowchart sebelum membuat blok.

Fokus utama bukan tampilan paling ramai, tetapi alur yang jelas, data yang aman, dan keputusan yang masuk akal.

Langkah Membuat Project di App Inventor

Gunakan langkah ini sebagai alur praktik di kelas. Guru bisa menghentikan presentasi di tiap langkah dan memberi waktu siswa mencoba.

1. Pilih ide app
2. Gambar flowchart
3. Susun Designer
4. Buat validasi Blocks
5. Buat keputusan data
6. Uji input salah

Checkpoint 1

Designer sudah punya input, tombol, output, dan Notifier.

Checkpoint 2

Blocks menolak data kosong atau data yang bukan angka.

Checkpoint 3

Aplikasi memberi saran berdasarkan kondisi data.

Rubrik Mini Project

Rubrik ini membantu siswa tahu kualitas aplikasi yang diharapkan.

Logika

Alur if-then-else jelas, tidak menghitung sebelum validasi, dan setiap kondisi punya hasil.

Design

Label, TextBox, tombol, dan output tertata. Pengguna tahu apa yang harus diisi.

Privacy

Aplikasi tidak meminta data yang tidak perlu dan tidak menampilkan data rahasia.
